Sesión 1

Introducción al curso y temas por aprender de HTML y CSS

Marzo 19, 2024 Laboratorio 2

Descripción de clase

Durante la sesión, nos familiarizamos con la interfaz de Visual Studio Code y aprendimos a crear y organizar archivos para nuestra página web. El profesor destacó la importancia de entender el lenguaje HTML como base fundamental para la estructura de nuestras páginas, explicando conceptos como etiquetas de inicio, atributos, valores y contenido.

Además, comenzamos a entender el diseño web a través de la analogía de "cajas apiladas", lo que nos ayudó a visualizar la jerarquía y relación entre las diferentes partes de una página. Empezamos a configurar nuestro primer código con la etiqueta !DOCTYPE, sentando las bases de la sesión explicada.

Sesión 2

Estructura y edición de plantilla index

Marzo 20, 2024 Laboratorio 2

Descripción de clase

En la segunda clase, nos adentramos en el manejo práctico de Visual Studio Code, editando el archivo Index de una plantilla predefinida. El profesor nos instruyó sobre la importancia de corregir errores de sintaxis y semántica, resaltando la jerarquía de etiquetas como h1 hasta h6 y su impacto en la estructura y relevancia del contenido. Aprendimos a utilizar elementos esenciales como Head para sugerencias, Title para el título de la página, Body para el contenido principal, así como P para párrafos y Ol y Ul para listas ordenadas y no ordenadas respectivamente. Además, nos introdujo al uso de hipervínculos con Href="#" y nos guió en el proceso de guardar y comprimir archivos para compartir nuestro trabajo.

Esta sesión nos proporcionó las bases necesarias para comenzar a diseñar y desarrollar paso a paso nuestras propias páginas web de manera efectiva, comprendiendo la importancia de la organización en la escritura de código HTML.

Sesión 3

Etiquetas: Apariencia de caracteres y espaciado / Sintaxis vs Semántica

Marzo 26, 2024 Laboratorio 2

Descripción de clase

En la tercera clase, el profesor nos introdujo a las etiquetas de apariencia de caracteres y de espacio en HTML, lo que amplió nuestra comprensión sobre el diseño y la presentación de contenido en páginas web. En el ámbito de la apariencia de caracteres, exploramos la etiqueta b para representar texto en negrita, i para texto en cursiva, sub para texto en subíndice, por ejemplo: 20091, útil para indicar pie de página u otras notas y sup ejemplo: 4 th.

En cuanto a la apariencia de espaciado, aprendimos sobre:
br / para realizar saltos de línea,
y hr / para insertar líneas horizontales, elementos que contribuyen a la estructuración visual y organización del contenido en la página.

Posteriormente, el profesor nos adentró en el concepto de semántica en HTML, resaltando la importancia de agregar significado y contexto al contenido mediante etiquetas como strong y em para dar énfasis y resaltar importancia, aportando claridad y relevancia al texto.

Sesión 5

Etiquetas Semánticas, Atributos y CSS en HTML

Abril 02, 2024 Laboratorio 2

Descripción de clase

Durante la primera hora de la clase, se dedicó tiempo a responder consultas y brindar asistencia personalizada a los compañeros para mejorar sus páginas web.

Posteriormente, el profesor amplió nuestro conocimiento sobre etiquetas semánticas en HTML, resaltando su importancia para enfatizar el contenido en los navegadores. Exploramos etiquetas como em para destacar contenido importante en cursiva, blockquote para citas largas con sangría, abbr para abreviaturas y acrónimos, cite para referencias a libros o grabaciones en cursiva, y dfn para términos o jergas. Además, se introdujo un nuevo tipo de lista con dl, dt y dd para definiciones y términos anidados.

También revisamos atributos en HTML, incluyendo los de imágenes como src, alt, title, height, width y align. Aprendimos a agregar videos en formato MP4 con atributos como src, poster, control, autoplay y muted loop. Se destacó la importancia de las etiquetas div> y span, y se brindó una introducción a CSS como herramienta para dar estilo a las páginas HTML. Finalmente, se explicaron las etiquetas necesarias para sincronizar CSS y HTML, adelantando el contenido de la próxima clase.

Sesión 6

Mis primeros ajustes con CSS

Abril 03, 2024 Laboratorio 2

Descripción de clase

En la clase 6, practicamos con CSS en una página de prueba, experimentando con cambios de color en títulos, párrafos, en elementos generales, así como con en elementos específicos mediante el uso de caracteres especiales. Además, exploramos la paleta de colores y aprendimos a ajustar la opacidad de los elementos utilizando la propiedad {background}.

Se nos enseñó a manipular el tamaño de letra con {font-size}, recordando que el tamaño predeterminado es 16px y se puede especificar en porcentaje o píxeles. También exploramos la propiedad {font-face} para especificar fuentes con derechos de autor y la importancia de tener una carpeta de fuentes.

Otras propiedades que abordamos incluyeron {font-family}, {font-weight} para negritas, {text-transform} para transformaciones de texto, y {text-align} para alineación de párrafos. Además, exploramos {border-width} y {border-style} para estilos de bordes, junto con otras opciones para personalizar aún más la apariencia de nuestras páginas web.

Sesión 7

Interacción con CSS / Conceptos básicos de Hosting y Dominio

Abril 09, 2024 Laboratorio 2

Descripción de clase

En esta clase, el profesor nos guió a través de la interacción con el puntero (mouse), destacando la posibilidad de generar transiciones de colores en botones para mejorar la experiencia del usuario. Además, nos enseñó la estructura necesaria para centrar una caja en nuestra página web, así como los conceptos y tipos de posicionamiento CSS, como (:static, :relative, :absolute, :fixed y :float, para ordenar las cajas de manera adecuada).

Continuando con el tema, exploramos el hosting y los dominios, donde aprendimos que un sitio web es un conjunto de páginas escritas en HTML y que un dominio es un nombre único asociado a una dirección IP.

Se destacó la importancia de elegir un dominio que refleje el nombre de la empresa de manera concisa, y se explicó el funcionamiento de la DNS como un directorio de IPs que permite encontrar las direcciones web. Finalmente, se presentaron diversas extensiones de dominio, como (.com, .net, .gob), y se ofreció una visión general sobre su uso y significado.
